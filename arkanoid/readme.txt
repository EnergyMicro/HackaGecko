Arkanoid game using touchless sensor to control.

This program demostrates touchless game control using Si1143 proximity sensor.

The project file is configured to be located in kits\EFM32WG_DK3850\examples directory, 
however it could be easily converted to other boards DK3x50 series.

This game is result of Hack-a-Gecko project and is not intended for production purposes.

Board:  Energy Micro EFM32WG-DK3850 Development Kit
Device: EFM32WG990F256
